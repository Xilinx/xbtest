##########################################################################
Platform definition JSON file
##########################################################################

********************************************************
Goal
********************************************************

You will now finalize and cross-check the platform definition JSON file (``xbtest_pfm_def.json``), by answering a series of questions.

.. important::
    It's highly recommended to track any modifications and/or result out of expected range.

    Please fill a problem report and add their tracking number within this page.

********************************************************
Prerequisite
********************************************************

You need to run all pre-canned tests once to answer to some of the questions.

.. warning::
    The first initial run of pre-canned tests will report errors as the platform definition JSON template contains standard thresholds (which you'll update while filling the checklist).

    If a default pre-canned test is failing, and it's because a nominal setting is wrong. then you should update the setting in platform definition JSON file.
