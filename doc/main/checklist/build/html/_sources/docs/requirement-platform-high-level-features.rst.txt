##########################################################################
Requirement - Platform high level features
##########################################################################

What are your expectations/requirements for xbtest?

Add your results to section :ref:`requirements-checklist` of your checklist.

********************************************************
Platform and card requirements
********************************************************

.. table:: Platform and card requirements description

    +--------------------------------------+------------------------------------------------------------------------------------------------------------+
    | Information                          | Description                                                                                                |
    +--------------------------------------+------------------------------------------------------------------------------------------------------------+
    | FPGA part                            | Part of the FPGA present on your card.                                                                     |
    +--------------------------------------+------------------------------------------------------------------------------------------------------------+
    | PCIe speed and width                 | PCIe Interface of your platform.                                                                           |
    +--------------------------------------+------------------------------------------------------------------------------------------------------------+
    | DMA type                             | For example: XDMA, QDMA or NoDMA.                                                                          |
    +--------------------------------------+------------------------------------------------------------------------------------------------------------+
    | DDR: speed, size and quantity        |                                                                                                            |
    +--------------------------------------+------------------------------------------------------------------------------------------------------------+
    | HBM: size, quantity and power rail   |                                                                                                            |
    +--------------------------------------+------------------------------------------------------------------------------------------------------------+
    | Host memory support                  |                                                                                                            |
    +--------------------------------------+------------------------------------------------------------------------------------------------------------+
    | P2P support                          |                                                                                                            |
    +--------------------------------------+------------------------------------------------------------------------------------------------------------+
    | GTs: type, quad, SLR location        | .. warning::                                                                                               |
    |                                      |     GT port numbering in v++ must align to the silkscreen or official port number on the physical board.   |
    |                                      |                                                                                                            |
    |                                      |       * Make sure to follow schematic numbering convention and avoid inversion.                            |
    +--------------------------------------+------------------------------------------------------------------------------------------------------------+

********************************************************
Power rails requirements
********************************************************

For each power rails of your card (e.g. ``12v_pex``, ``3v3_pex``, ``12v_aux``, ``vccint``), specify:

  * Card critical limits.
  * Target power consumption (only applicable if different to critical power limit).
